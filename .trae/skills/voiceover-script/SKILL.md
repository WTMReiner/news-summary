# 配音脚本生成器

该功能可基于Reiner AI每日信息差报告生成专业的视频内容配音脚本。

## 触发方式

- 用户发送：使用voiceover-script将[文件路径]生成配音脚本

## 使用方法

1.  **提取内容**：
    该功能首先使用`extract_news.py`从提供的HTML文件中提取新闻内容。
    
    ```bash
    python .trae/skills/Reiner-ai-daily/voiceover-script/extract_news.py <HTML文件路径>
    ```

2.  **生成脚本**：
    智能体基于提取的JSON内容生成配音脚本。
    
    **默认参数**：
    -   **风格**：专业、清晰、简洁精炼，剔除无关冗余信息，只传递观众需要的核心内容，避免无意义的铺垫。
    -   **语言**：简体中文
    -   **结构**：
        -   **开头**：用4-8个字讲出关键新闻，激发观众的探究欲。
        -   **主体**：从每条新闻中筛选关键信息（优先选择重大科技/AI新闻）。
    -   **格式**：
        -   `[画面：...]`（镜头/画面提示）
        -   `(语气：...)`（语气/语速提示）
        -   "配音文本"
